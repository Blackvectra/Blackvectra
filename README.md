## About

Security engineer focused on production blue-team operations across managed service
environments. Founder of NextLayerSec, an independent consulting practice delivering
email security, identity hardening, and Microsoft 365 defense frameworks for small
and mid-market organizations.

Areas of focus:
- Email authentication and transport hardening (SPF / DKIM / DMARC / MTA-STS / DNSSEC)
- Microsoft 365 tenant security baselines and assessment
- Identity protection across Entra ID and Conditional Access
- Defense-in-depth playbooks aligned to NIST SP 800-53 and CIS Controls v8
